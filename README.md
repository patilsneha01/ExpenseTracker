# Expense Tracker (Qt C++)

A desktop-based Expense Tracker application built using **Qt Widgets** and **C++**.  
This project helps you track daily expenses, categorize them, and view them in a clean table format.

---

## Features

- Add expenses with **amount, category, Date and description**
- Auto-increment **Sr No** for each expense
- Category selection via **dropdown**
- Modern and colorful **UI styling**
- Alternating row colors for better readability
- Professional project structure with **OOP concepts**

---

## Technologies Used

- **C++11**  
- **Qt Widgets**  
- **QTableWidget, QPushButton, QLineEdit, QComboBox**  
- **CMake / Qt Creator**
